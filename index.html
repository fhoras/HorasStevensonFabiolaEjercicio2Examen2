Ejercicio 2 - JavaScript

Realiza las funciones de javascript necesarias para que dado un input y un boton en la pagina html, permita al usuario colocar un numero en el campo input y al presionar el boton muestre en la pagina tantas cajas como indica el numero ingresado y con los estilos de alto, ancho y color de fondo de la caja colocado con código javascript. Mientras quepan las cajas deben estar una al lado de otra. Nota: adjunta el enlace a github del repositorios del código el cual debes adjuntar a la entrega. El nombre del archivo debe tener el siguiente formato apellido_nombre_ej2.html
